`paicku help`
=============

Display help for paicku.

* [`paicku help [COMMAND]`](#paicku-help-command)

## `paicku help [COMMAND]`

Display help for paicku.

```
USAGE
  $ paicku help [COMMAND...] [-n]

ARGUMENTS
  COMMAND...  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for paicku.
```
